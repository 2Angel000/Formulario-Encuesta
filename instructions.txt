1. Debes tener un título de página en un elemento h1 con un id de title

2. Debe tener una breve explicación en un elemento p con un id de description

3. Debe tener un elemento de formulario con un id de survey-form

4. Dentro del elemento formulario, debe introducir su nombre en un campo de entrada que tiene un id de nombre y un tipo de texto

5. Dentro del elemento de formulario, se requiere que introduzca su correo electrónico en un campo de entrada que tiene un id de correo electrónico

6. Si introduce un correo electrónico que no está formateado correctamente, verá un error de validación HTML5

7. Dentro del formulario, puedes introducir un número en un campo de entrada que tiene un id de número

8. Si introduces un número que no es de este tipo, verás un error de validación de HTML5

9. Si introduces números fuera del rango de la entrada numérica, definidos por los atributos min y max, verás un error de validación HTML5

10. Para los campos de entrada de nombre, correo electrónico y número, puedes ver los elementos de etiqueta correspondientes en el formulario, que describen el propósito de cada campo con los siguientes ids: id="name-label", id="email-label", y id="number-label"

11. Dentro del elemento del formulario, debes tener un elemento desplegable de selección con un id de dropdown y al menos dos opciones para elegir

12. Dentro del elemento del formulario, debes tener un elemento desplegable de selección con un id de dropdown y al menos dos opciones para elegir

13. Dentro del elemento formulario, puedes seleccionar una opción de un grupo de al menos dos botones de radio que se agrupan utilizando el atributo name

14. Dentro del elemento formulario, puedes seleccionar varios campos de una serie de casillas de verificación, cada una de las cuales debe tener un atributo value

15. Dentro del elemento formulario, se presenta un área de texto para comentarios adicionales

16. Dentro del elemento formulario, se presenta un botón con id de submit para enviar todas las entradas
